## 概要

MindHub は、深く構造化された作業のために設計された階層型タスク＆ノートアプリです。ワークスペース全体がツリー構造です — プロジェクトにはフェーズがあり、フェーズにはタスクがあり、タスクにはサブタスクがあります。macOS 上でネイティブに動作し、内部には本物のローカル SQLite データベースを使用しています。

## 機能

- **多段階ネスト** — 無制限の階層レベルで作業を構造化。任意のブランチをキー一つで折り畳み。
- **Spotlight 一括編集** — 複数のアイテムを選択し、ステータス・タグ・期日を一度に更新。
- **キーボード駆動** — すべての操作にショートカットあり。キーボードから手を離さずに作成・移動・インデント・完了が可能。
- **ローカル SQLite データベース** — データはデバイス上に保存。オフラインで動作し、即座に起動、ネットワーク依存なし。
- **200 ステップの元に戻す/やり直し** — ファイルごとにディスクへ永続化。アプリ再起動後も機能します。
- **30 日間保持のゴミ箱** — 削除したアイテムは復元可能で、完全には消えません。
- **18 種類の手作りテーマ** — 9 つの温かいライト、9 つの深いダーク。
