## 概览

MindHub 是一款层级式任务与笔记应用，专为深度、结构化的工作而生。整个工作区是一棵树——项目下有阶段，阶段下有任务，任务下有子任务。原生运行于 macOS，底层采用真正的本地 SQLite 数据库。

## 功能

- **多级嵌套** — 跨无限层级组织工作；一键折叠任意分支。
- **Spotlight 批量编辑** — 选中多个条目，一次性更新状态、标签或截止日期。
- **键盘驱动** — 每个操作都有快捷键。无需离开键盘即可完成创建、移动、缩进和完成。
- **本地 SQLite 数据库** — 数据存储在你的设备上。离线可用，即时启动，无需网络。
- **200 步撤销/重做** — 按文件持久化到磁盘，应用重启后依然有效。
- **30 天保留的回收站** — 已删除条目可恢复，不会永久消失。
- **18 款手工精制主题** — 9 款温暖浅色，9 款深邃暗色。
