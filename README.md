## Overview

MindHub is a hierarchical task and note app built for deep, structured work. Your entire workspace is a tree — projects have phases, phases have tasks, tasks have subtasks. It runs natively on macOS with a real local SQLite database underneath.

## Features

- **Multi-level nesting** — Structure work across unlimited hierarchy levels; collapse any branch with one keystroke.
- **Spotlight batch editing** — Select multiple items and update status, tags, or due date in one shot.
- **Keyboard-driven** — Every action has a shortcut. Create, move, indent, complete without leaving the keyboard.
- **Local SQLite database** — Your data lives on your device. Works offline, launches instantly, no network dependency.
- **200-step undo / redo** — Persisted to disk, per file. Works across app restarts.
- **Trash with 30-day retention** — Deleted items are recoverable, not gone.
- **18 handcrafted themes** — Nine warm lights, nine deep darks.
