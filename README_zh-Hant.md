## 概覽

YouMindHub 是一款層級式任務與筆記應用程式，專為深度、結構化的工作而生。整個工作區是一棵樹——專案下有階段，階段下有任務，任務下有子任務。原生執行於 macOS，採用本地儲存。

## 功能

- **多層嵌套** — 跨無限層級組織工作；一鍵折疊任意分支。
- **Spotlight 批次編輯** — 選取多個項目，一次性更新狀態、標籤或截止日期。
- **鍵盤驅動** — 每個操作都有快捷鍵。無需離開鍵盤即可完成建立、移動、縮排和完成。
- **本地儲存** — 離線可用，即時啟動，無需網路（僅登入和建立 Project 需要）。
- **200 步撤銷/重做** — 按檔案持久化到磁碟，應用程式重新啟動後依然有效。
- **30 天保留的資源回收桶** — 永久刪除前可復原。
- **18 款手工精製主題** — 9 款溫暖淺色，9 款深邃暗色。
- **雲端同步** — 在所有裝置間同步資料。無論在哪裡，隨時隨地繼續你的專案。

## 系統需求

- Apple Silicon 或 Intel Mac
