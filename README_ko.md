## 개요

MindHub는 깊고 체계적인 작업을 위해 설계된 계층형 작업 및 노트 앱입니다. 전체 작업 공간이 트리 구조입니다 — 프로젝트에는 단계가 있고, 단계에는 작업이 있으며, 작업에는 하위 작업이 있습니다. macOS에서 네이티브로 실행되며 내부에 진짜 로컬 SQLite 데이터베이스를 사용합니다.

## 기능

- **다단계 중첩** — 무제한 계층 수준에서 작업을 구조화하고, 단 하나의 키로 원하는 브랜치를 접을 수 있습니다.
- **Spotlight 일괄 편집** — 여러 항목을 선택하고 상태, 태그 또는 마감일을 한 번에 업데이트할 수 있습니다.
- **키보드 중심** — 모든 동작에 단축키가 있습니다. 키보드를 벗어나지 않고 생성, 이동, 들여쓰기, 완료가 가능합니다.
- **로컬 SQLite 데이터베이스** — 데이터가 내 기기에 저장됩니다. 오프라인에서도 작동하고, 즉시 실행되며, 네트워크 의존성이 없습니다.
- **200단계 실행 취소/다시 실행** — 파일별로 디스크에 저장됩니다. 앱 재시작 후에도 작동합니다.
- **30일 보존 휴지통** — 삭제된 항목은 복구할 수 있으며 영구적으로 사라지지 않습니다.
- **18가지 수제 테마** — 따뜻한 밝은 테마 9가지, 깊은 어두운 테마 9가지.

## 요구 사항

- macOS 14 이상
- Apple Silicon 또는 Intel Mac
